## Introduction
QMS-Flow is a software engineering project developed to digitize and automate ISO 9001:2015 compliance workflows. The system replaces manual, paper-based processes with a structured digital platform to manage audits, document control, and customer feedback.

## Project Scope
The system is divided into four main modules designed to ensure quality standards are met:

* **Document Control:** Managing the lifecycle of SOPs and quality documents.
* **Audit Management:** Scheduling and tracking internal quality audits.
* **Customer Complaints:** Logging feedback and linking them to corrective actions.
* **Management Review:** Providing data-driven insights for strategic decision-making.

## System Architecture
This project follows a **Layered Architectural Style**, separating concerns into distinct tiers:
* **Presentation Layer:** The user interface for interacting with the system.
* **Domain Layer:** The core business logic and compliance rules.
* **Data Layer:** Relational database management for storing quality records.

## Technical Stack
* **Development:** PHP, JavaScript, and HTML/CSS
* **Database:** MySQL
* **Modeling:** UML Methodology (Use Case and Logical Views)
